# 新UI设计 - 开始界面（机型选择）

## 界面展示

```
┌─────────────────────────────────────────┐
│  ━━━ 路由器刷机工具 v1.0 ━━━       │
│  工作节点：router-flash-01              │
├─────────────────────────────────────────┤
│                                         │
│  请选择要刷机的路由器型号：           │
│                                         │
│  [ 1 ]  CR660X 小米/联通路由器     │
│  [ 2 ]  JGC Q10/Q20 路由器         │
│                                         │
├─────────────────────────────────────────┤
│  已完成: 0 台                         │
└─────────────────────────────────────────┘

请输入数字 [1-2]：
```

## 优化点

1. **去掉复杂标题**："半自动流水线程序"太技术，改成"路由器刷机工具"
2. **简化描述**：把"TR660X"去掉，工人只关心型号
3. **统一格式**：使用 `[ ]` 表示选项，简洁清晰
4. **底部状态**：显示已完成的数量，工人知道进度
5. **输入提示**：明确告诉输入范围 `[1-2]`

## 交互逻辑

```
用户输入 → 验证是否为 1 或 2
  ├─ 有效 → 进入对应机型界面
  └─ 无效 → 重新提示输入（不改界面）
```

## 工人操作指南

1. **CR660X**：小米盒子、联通定制路由器选这个
2. **JGC**：其他型号选这个
3. **不确定**：看路由器背面的型号贴纸
