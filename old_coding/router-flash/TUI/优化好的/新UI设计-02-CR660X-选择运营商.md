# 新UI设计 - CR660X 选择运营商

## 界面展示

```
┌─────────────────────────────────────────┐
│  CR660X 路由器刷机                 │
├─────────────────────────────────────────┤
│                                         │
│  请选择路由器运营商：                 │
│                                         │
│  [ 1 ]  移动 / 电信                   │
│         └─ 需要输入路由器密码         │
│  [ 2 ]  联通                           │
│         └─ 不需要密码，自动刷机         │
│                                         │
├─────────────────────────────────────────┤
│  提示：联通版本会恢复出厂设置     │
└─────────────────────────────────────────┘

请输入数字 [1-2]：
```

## 优化点

1. **用缩进说明子项**：`└─` 清晰表示说明内容
2. **突出重点**：需要密码的用加粗显示
3. **提示简洁**：联通恢复出厂用一句话说明即可
4. **去掉废话**：不需要解释"破解脚本"等技术细节

## 交互逻辑

```
用户输入 → 验证是否为 1 或 2
  ├─ [1] 移动/电信 → 进入密码输入界面
  └─ [2] 联通     → 直接开始刷机
```

## 工人操作指南

1. **移动/电信**：需要先知道路由器密码，在路由器背面贴纸上
2. **联通**：可以全自动刷机，准备好路由器即可
3. **不知道类型**：看路由器外壳或登录界面，有运营商标识
