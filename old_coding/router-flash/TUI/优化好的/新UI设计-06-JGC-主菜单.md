# 新UI设计 - JGC 主菜单

## 界面展示

```
┌─────────────────────────────────────────┐
│  JGC Q10/Q20 路由器刷机           │
├─────────────────────────────────────────┤
│                                         │
│  在线路由器：192.168.10.1           │
│  下载服务：● 运行中 (123.2:8080)   │
│                                         │
│  请选择刷机步骤：                   │
│                                         │
│  [ 1 ] 步骤 1：官放→PDCN           │
│         └─ 更换路由器系统              │
│  [ 2 ] 步骤 2：PDCN→引导程序       │
│         └─ 刷入启动引导              │
│  [ 3 ] 步骤 3：引导→最终系统       │
│         └─ 刷入最终系统              │
│                                         │
├─────────────────────────────────────────┤
│  已完成: 0 台                         │
└─────────────────────────────────────────┘

请输入数字 [1-3]：
```

## 优化点

1. **状态一目了然**：顶部直接显示在线 IP 和服务状态
2. **简化步骤名称**：用箭头 `→` 表示转换，比文字更直观
3. **一句话说明**：每个步骤一句话说明干什么
4. **服务状态可视化**：`●` 表示运行，`○` 表示停止
5. **缩进说明**：用 `└─` 表示补充说明

## 状态指示符

| 符号 | 含义 | 颜色 |
|------|------|------|
| ● | 运行中 | 绿色 |
| ○ | 已停止 | 灰色 |
| ❌ | 错误/失败 | 红色 |
| ✅ | 成功/在线 | 绿色 |
| ⚠️ | 警告 | 黄色 |

## 交互逻辑

```
JGC 主菜单 →
  ├─ [1] 步骤1 → 检测密码 → 刷入PDCN → 自动进入步骤2
  ├─ [2] 步骤2 → 检查下载服务 → 刷入引导 → 自动进入步骤3
  └─ [3] 步骤3 → 刷入最终系统 → 统计库存
```

## 工人操作指南

1. **第一次刷机**：从步骤 1 开始，按顺序完成
2. **已经刷过步骤1**：直接选步骤 2 或 3
3. **下载服务必须运行**：如果显示 `○ 已停止`，会自动启动
4. **不确定进行到哪**：看在线路由器 IP 判断
   - 2.x/10.x：还没开始，选步骤 1
   - 123.x：已经到步骤 2，选步骤 2 或 3
   - 1.x：已经到步骤 3，选步骤 3
