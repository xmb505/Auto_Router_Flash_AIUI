# 开始界面 - 路由器机型选择

## 界面展示
```
-----路由器刷机半自动流水线程序-----
现在的窗口编号为：（显示 hostname，如 router-flash-01）

输入数字选择刷机机型
1. CR660X / TR660X
2. JGC Q10/Q20

你输入的数字：

-----Powered by Haku Studio 2026-----
```

## 逻辑说明

### 输入验证
- 只接受输入 `1` 或 `2`
- 其他输入：提示重新输入

### 流程跳转
- 输入 `1` → 进入 CR660X 刷机界面
- 输入 `2` → 进入 JGC 刷机界面

### 主机名显示
- 使用 `hostname` 命令获取当前主机名
- 通常格式为 `router-flash-XX`，方便识别不同刷机节点

## 小提示
1. 暂不支持同时刷两种路由器，选择后进入对应流程
2. 主机名可以由用户在 PVE 层面设置，用于区分多个刷机节点
3. 此界面为程序入口，刷机过程中不会再次返回
