# CR660X 选择运营商界面

## 界面展示
```
-----CR660X/TR660X 批量刷机模式-----
所处模式：（1：刷机模式）

选择运营商：（用户输入的数字）
1：中国移动/中国电信
2：中国联通


-----Powered by Haku Studio 2026-----

（提示：联通是恢复出厂设置然后我自己设置密码自己刷）
```

## 逻辑说明

### 输入验证
- 只接受输入 `1` 或 `2`
- 其他输入：提示重新输入

### 运营商差异处理
**选择 1：中国移动/中国电信**
- 需要**手动输入路由器管理密码**（密码明文显示，不遮掩）
- 路由器会处于官方固件状态，需要破解 SSH 并刷入 bootloader + kernel

**选择 2：中国联通**
- **不需要输入密码**（恢复出厂后无密码）
- 自动执行刷机流程

### 流程跳转
- 输入 `1` → 进入「检测IP + 输入密码」流程 → 执行 `test_login1.sh`
- 输入 `2` → 直接进入「检测IP + 自动刷机」流程 → 执行 `unicom_flash.sh`

## 小提示
1. 联通版本恢复出厂后默认无密码，流程更简单
2. 移动/电信版本需要管理员密码用于登录 Web UI 开启 SSH
3. **密码输入不要遮住**（用户明确要求明文），方便确认和调试
4. 密码错误会导致 SSH 开启失败，会提示查看日志或重试
